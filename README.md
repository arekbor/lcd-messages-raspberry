# lcd-messages-raspberry

## inputs for 4-bit display

- VSS - ground
- VDD - VDC (5V)
- VO - ground
- RS - PIN11
- RW - ground
- E - PIN10
- D4 - PIN0
- D5 - PIN1
- D6 - PIN2
- D7 - PIN3
